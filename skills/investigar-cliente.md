# Skill: Investigar cliente antes de reunión

## Cuándo usarla
Cuando entra un nuevo lead o cliente y tienes una llamada/reunión próxima.

## Pasos
1. Buscar la web del cliente y extraer: qué hace, tamaño, sector
2. Identificar dolores probables que podemos resolver con automatizaciones o desarrollo web
3. Ver si hay casos de éxito similares en nuestra cartera
4. Proponer 2-3 soluciones concretas (web, automatización n8n, IA)
5. Crear un brief en /projects/[nombre-cliente].md con:
   - Resumen del negocio
   - Dolores detectados
   - Soluciones a proponer
   - Preguntas clave para la reunión
   - Ticket estimado

## Cómo activarla
"Tengo una reunión con [nombre empresa], prepárame el brief"
