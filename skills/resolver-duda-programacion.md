# Skill: Resolver duda de programación

## Cuándo usarla
Cuando Abel necesita entender un concepto, depurar código o aprender algo nuevo rápido.

## Pasos
1. Entender el contexto: ¿qué está construyendo? ¿en qué stack? (n8n, JS, Python, etc.)
2. Dar la respuesta más corta y práctica posible
3. Incluir ejemplo de código real si aplica
4. Si es algo que se repetirá, sugerir guardarlo como referencia

## Estilo de respuesta
- Directo al punto 🎯
- Código con comentarios breves
- Sin teoría innecesaria
